# BiasBreaker Backend

A backend service for the BiasBreaker project that helps detect and mitigate bias in AI systems.

## Prerequisites

- Node.js (v14 or higher)
- npm or yarn
- Git

## Installation

```bash
# Clone the repository
git clone https://github.com/drjayaswal/biasbreaker-backend.git

# Navigate to the project directory
cd biasbreaker-backend

# Install dependencies
npm install
```

## Environment Setup

```bash
# Create a .env file
cp .env.example .env

# Update .env with your configuration
```

## Running the Backend

```bash
# Development mode
npm run dev

# Production mode
npm run build
npm start
```

## Running with Frontend

```bash
# Ensure backend is running on http://localhost:5000 (or configured port)

# In another terminal, clone and run frontend
git clone https://github.com/drjayaswal/biasbreaker-frontend.git
cd biasbreaker-frontend
npm install
npm start
```
